# Detection of Cyber Abuse Language (Roman Urdu)

This project focuses on detecting cyber abusive language written in Roman Urdu
using Machine Learning techniques.

## Features
- Roman Urdu text preprocessing
- TF-IDF vectorization
- Multiple ML models (Logistic Regression, SVM, Random Forest)
- Jupyter Notebook based training
- Flask-based application for prediction

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Flask

## Models
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest
